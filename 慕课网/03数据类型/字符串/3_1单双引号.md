# python 数据类型

## 字符串

- `str`字符串

  - 单引号 - 'abcd'
  - 双引号 - ''abcd''
  - 三引号 - '''abcd'''

  ```py

    a = 'let's go '
    a1 = 'let"s go '

    a2 = 'let\'s go '  # \ 转义字符
    a3 = "let's go"

  ```

  ```py
  打印结果:  报错
            let"s go
            let's go
            let's go
  ```
