# 第十章 正则表达式 JSON

## 字符集

- **[ ]** ,(或,匹配 x 或 y)
- **^** ,(取反,匹配 不是 x 和 不是 y)
- **-** ,(全部匹配,取 a-z)

```py
  例子:

  import re


  b = 'abc,acc,adc,aec,afc,ahc'

  s1 = re.findall('a[cf]c', b)
  print(s1)

  s2 = re.findall('a[^cfd]c', b)
  print(s2)

  s3 = re.findall('a[c-f]c', b)
  print(s3)

```

```py
打印结果:
['acc', 'afc']
['abc', 'aec', 'ahc']
['acc', 'adc', 'aec', 'afc']

```
