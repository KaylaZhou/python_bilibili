## 初始编码

- ASCII, 8 位=一个字节, 2\*\*8
- unicode ,万国码, 32 位=4 个字节,2\*\*32
- UTF-8,unicode 的压缩版,最少用 1 字节=8 位,最多用 4 字节=32 位表示.中文:3 字节=24 位表示.

### python 解释器码

- python 2: ascii,运行时添加文件头:
  ```
  # -*-coding:utf-8 -*-
  print('你好')
  ```
- python 3: utf-8

### 文件编码

- 编写文件时,保存文件用 UTF-8 保存
- 编码+解码一致(以什么编码保存,用什么编码打开,否则会乱码)
