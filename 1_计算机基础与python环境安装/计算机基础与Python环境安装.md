## 学习编程语言的常见步骤

### 安装 解释器/编译器/虚拟机(python3)

- 下载解释器

  官网下载:<www.python.org>

- 安装 python3.6.8
- 检查 python3.6.8 是否安装成功
- 通过电脑终端检查:

```shell
python -V  #会打印版本
```
